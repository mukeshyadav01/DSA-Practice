# DSA Practice (C++)

Practice programs to learn Data Structures and Algorithms fundamentals in C++, organized topic-wise.

## Structure

- **[Array](./Array)** – array traversal, searching (linear search)
- **[forloop](./forloop)** – for loop based programs
- **[function](./function)** – function definitions and usage
- **[if](./if)** – conditional statement programs
- **[Pattern_printing](./Pattern_printing)** – star/number pattern programs using nested loops
- **[While & Do while loop](./While%20%26Do%20while%20loop)** – while and do-while loop examples

## How to run

Each folder contains standalone `.cpp` files. To compile and run any file:

```bash
g++ filename.cpp -o output
./output
```

## Notes

This repository is a work-in-progress as I learn C++ and DSA concepts step by step. Each `.cpp` file has a comment header explaining what it does.
